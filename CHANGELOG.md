# Changelog

Все важные изменения проекта Markdown Document Converter документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [2.1.0] - 2024-11-18

### Добавлено
- Система версионирования с константой APP_VERSION
- Динамическое отображение версии в футере страницы
- Вывод версии и даты сборки в консоль браузера
- Детальное логирование обработки Mermaid диаграмм
- Файл CHANGELOG.md для истории версий

### Изменено
- Полная переработка генерации PDF с использованием jsPDF API
- Шрифт изменен с Times на Helvetica для поддержки кириллицы
- Удален селектор стиля (всегда профессиональный академический)
- Обновлен README с подробным changelog

### Исправлено
- Поддержка кириллицы в PDF документах
- Отображение границ таблиц в PDF (через jsPDF-AutoTable)
- Рендеринг Mermaid диаграмм в PDF
- Заголовки H1 теперь начинаются с новой страницы
- Диаграммы автоматически масштабируются на одну страницу
- Ошибка "Invalid argument passed to jsPDF.scale"
- Обработка пустых заголовков и недействительных canvas

### PDF Генерация
- Шрифт: Helvetica (с поддержкой Unicode/Кириллицы)
- Размер основного текста: 12pt
- Заголовки: Title 18pt, H1 16pt, H2 14pt, H3 12pt
- Поля: 25mm со всех сторон
- Таблицы: с границами и чередующимися строками
- Код: светло-серый фон (#f6f8fa)
- Диаграммы: автомасштабирование, помещаются на одной странице

## [2.0.0] - 2024-11

### Добавлено
- Статическая веб-версия конвертера без серверной части
- Конвертация Markdown в PDF через jsPDF и html2canvas
- Конвертация Markdown в DOCX через docx.js
- Поддержка YAML front matter для метаданных
- Подсветка синтаксиса кода (Highlight.js)
- Рендеринг Mermaid диаграмм
- Предпросмотр документа
- Drag & drop загрузка файлов
- Выбор формата вывода (PDF/DOCX)
- Выбор стиля оформления
- GitHub Actions для автодеплоя
- Полностью клиентская обработка (файлы не загружаются на сервер)

### Markdown Функции
- Заголовки (H1-H6)
- Параграфы и форматирование текста
- Списки (упорядоченные и неупорядоченные)
- Таблицы
- Ссылки
- Изображения
- Цитаты
- Блоки кода с подсветкой синтаксиса
- Горизонтальные линии

### Технологии
- markdown-it v13.0.2
- Highlight.js v11.9.0
- jsPDF v2.5.1
- jsPDF-AutoTable v3.5.31
- html2canvas v1.4.1
- docx.js v8.5.0
- file-saver v2.0.5
- Mermaid v10.6.1
- js-yaml v4.1.0

## [1.0.0] - 2024-10

### Добавлено
- Первоначальная версия (Python CLI)
- Конвертация Markdown в PDF и Word
- Поддержка базового синтаксиса Markdown
- Веб-версия на Flask
